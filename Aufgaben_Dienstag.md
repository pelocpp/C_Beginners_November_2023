# Aufgaben zu Unterprogrammen

### Aufgabe 1 - Gerade oder Ungerade

Schreiben Sie ein Unterprogramm `isEven`,
das einen `int`-Parameter besitzt
und als Ergebnis die Informtation zurückliefert,
ob der Wert gerade oder ungerade ist.

Ein Rückgabewert von `1` soll dabei für *gerade*
und `0` für *ungerade* stehen.


### Aufgabe 2 - Größere zweier Zahlen

Schreiben Sie ein Unterprogramm `maximum`,
das zwei `int`-Parameter besitzt
und den größeren Wert der zwei Parameter
als Ergebnis des Unterprogramms zurückliefert.



### Aufgabe 3 - Größere dreier Zahlen

Schreiben Sie ein Unterprogramm `maximum`,
das drei `int`-Parameter besitzt
und den größten Wert der drei Parameter
als Ergebnis des Unterprogramms zurückliefert.


### Aufgabe 4 - Teiler einer Zahl

Schreiben Sie ein Unterprogramm `wieOft`,
das zwei `int`-Parameter besitzt.
Der erste Parameter steht für eine beliebige Zahl,
der zweite Parameter für einen Teiler.

Die Funktion berechnet, 
wie oft der Teiler in der Zahl enthalten ist.

*Beispiel*:

Ein Aufruf von `wieOft (36, 3);`
liefert den Wert 2 zurück.
